🚨 No se puede iniciar el servidor

Opciones disponibles:

Opción 1: Usar servicios online (RECOMENDADO)

Puedes desplegar tu aplicación en servicios gratuitos que soportan Node.js:

  • Vercel: vercel.com
  • Netlify: netlify.com
  • Railway: railway.app
  • Render: render.com

Opción 2: Instalar Node.js (rápido)

Descarga e instala Node.js desde:

⬇️ Descargar Node.js

Después de instalarlo, ejecuta en la terminal:

npm install
node server/index.js

Opción 3: Probar el frontend sin servidor

Puedes abrir el archivo directamente, pero las notificaciones NO funcionarán sin un servidor backend.

Ver Frontend (solo visual)
